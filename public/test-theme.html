Light/Dark Mode Transitions

This page demonstrates smooth transitions when switching between light and dark themes. Click the theme toggle button in the top-right corner to see the animations in action.

Features

  • Smooth color transitions (300ms)
  • Background color animations
  • Text color transitions
  • Border color changes
  • Hover effects on the toggle button
  • Accessibility features

Code Example

/* CSS Transition Example */ body { transition: background-color 0.3s ease, color 0.3s ease; } .dark { --theme: rgb(29, 30, 32); --content: rgb(196, 196, 197); }

Animation Details

The theme switching uses CSS transitions with a duration of 300ms and an ease timing function. All color-related properties (background-color, color, border-color) are animated smoothly.
